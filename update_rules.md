# 變更記錄更新規範 (Changelog Rules)
當我要求「更新 README」或「記錄變更」時，請執行以下動作：

1. **定位區塊**：找到 `README.md` 中的 `## 🛠 更新日誌 (Update Log)` 或 `## 建立與更新日誌` 區塊。
2. **自動日期**：使用今天的日期作為標題，格式為 `### [YYYY-MM-DD]`。
3. **格式標準化**：
   - 使用 `- ` 或 `- [x] ` 作為清單開頭。
   - 項目類別需加粗體，例如：**功能改進**、**安全性修正**、**UI 調整**。
4. **寫入檔案**：直接編輯 `README.md` 並儲存，不要只在對話框回覆文字。
